# Excel Data Cleaning & Formatting

## Task Description
This project focuses on cleaning and formatting a real-world dataset using Microsoft Excel.

## Dataset Used
Netflix Movies and TV Shows dataset (CSV format)

## Tools Used
- Microsoft Excel

## Steps Performed
- Opened CSV dataset and validated headers
- Identified and handled missing values
- Removed duplicate records
- Standardized text using TRIM, PROPER, UPPER
- Corrected date and numeric formats
- Created a Cleaned_Data sheet
- Added Data_Quality_Notes column

## Files Included
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv

## Outcome
Produced a clean, structured dataset ready for analysis and visualization.
